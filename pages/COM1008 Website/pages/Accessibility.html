Accessibility

In this website, I have made sure I have designed and created every element taking others into account. This particularly includes the disabled. I have made sure that all text is large and easy to read for the partially blind and that all images include alternative texts for screen readers.

It also allows any other disability to perceive, understand, navigate and interact with the website in a way that they could normally. This ensures that the disabled will not feel neglected whilst using the website.

This website also adheres to the web standards supplied by the World Wide Web Consortium. It also has been designed to work for the color blind. The colours chosen have all been picked to provide contrasts that will not effect the colour blind and to allow them to see the website just the same as any other person.
