COM1008 | Web and Internet Technology

This module is worth 10 credits in the first level of the degree and is only taught in the first semester. As web and the Internet is now an essential need in many people's lives, it is only fair to teach students of what makes the Internet work and how to implement it. In this module, students learn the three fundamental languages used in websites: HTML, CSS and JavaScript. They also get told of the origins of the internet and how the world wide web works.
The assessment for this module takes forms of 2 MOLE quizzes, and a large website project showing off skills taught in the module.
Examples of things you will cover are:

  • Programming in HTML, CSS and JavaScript
  • Responsive Web Design
  • Web standards
  • Design and accessibility
  • Graphics, Canvas and SVG
