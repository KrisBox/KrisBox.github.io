COM1001 | Introduction to Software Engineering

This module is worth 20 credits in the first level of the degree. It introduces the basics needed to be a good software engineer. It covers the main steps in the process of developing systems all the way from the requirements analysis to the testing and implementation stages. A major part the module is a group project, enhancing the skills taught in lectures.
The assessment for this module takes forms of MOLE quizzes, problem sheets and the large team project. It is mostly focused on team building skills that will be essential for later modules in the degree.
Examples of things you will cover are:

  • The Software Lifecycle
  • Project Management (Kanban, Scrum etc)
  • Version Control
  • Programming in Ruby
  • Software testing and maintenance
  • Database design and implementation

Sessions are given in forms of lectures and lab sessions enhancing the content learnt.
