COM1003 | Java Programming

This module is worth 20 credits in the first level of the degree. It introduces the basic programming concepts that the students will need throughout the degree program. Students are encouraged to practise programs shown in the lectures and the emphasis of writing well-structured and readable programs is enforced to gain good habits for the students programming life.
The assessment for this module takes forms of MOLE quizzes, small assignments and various large assignments.
Examples of things you will cover are:

  • Basics of the Java programming language
  • The use of input and output in Java
  • The use of Arrays and Enums in Java
  • Object Orientated Programming
  • Use of the Java API and Swing
  • How to create javadocs for your own programs

Sessions are given in forms of lectures and lab sessions enhancing the content learnt.
