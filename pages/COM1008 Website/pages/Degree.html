My Degree

COM1001 | Introduction to Software Engineering

This module introduces concepts that are needed to develop software systems in teams all the way from analysis to design and testing. It encourages teamwork between students by a major web-based system accounting to a large part of the module marks.

COM1002 | Foundations of Computer Science

This module introduces key mathematical concepts and techniques that are of foundational importance to computing. It enhances student's logical thinking whilst preparing students for future courses and work.

COM1003 | Java Programming

This module introduces the basic programming techniques needed by the students through the Java language. Students learn about the basic structures of a program while later going onto more advanced topics such as object-oriented programming.

COM1005 | Machines and Intelligence

This module introduces the basic concepts of Artificial Intelligence. It begins talking about the history of AI and an introduction to popular AI topics such as visual processing and machine learning.


COM1006 | Devices and Networks

This module introduces the basic concepts of how computer architecture works including digital logic, assembly programming and computer arithmetic. All this information leads to how the CPU works down to its minimum.

COM1008 | Web and Internet Technology

This module introduces how to create responsive websites using the popular HTML, CSS, and JavaScript languages. It also introduces the ideas of networking protocols and how information is sent across the Internet.
